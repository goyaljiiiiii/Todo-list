# To-Do List

A simple and interactive To-Do List application to help you stay organized and manage your tasks efficiently. This project allows users to add, edit, and delete tasks. It offers an intuitive interface to keep track of your daily activities, helping you stay productive.

## Features

* **Add Tasks:** Easily add new tasks to your list.
* **Edit Tasks:** Update tasks as needed.
* **Delete Tasks:** Remove completed or unwanted tasks.
* **Mark Tasks as Completed:** Track the progress of your tasks.

## Technologies Used

* **HTML:** To structure the app.
* **CSS:** For styling and layout.
* **JavaScript:** For interactive functionality.

## Getting Started

Clone the repository:

git clone https://github.com/yourusername/todo-list.git


Open `index.html` in your browser to start using the app.
